## 🚀 Kurulum

Projeyi yerel makinenizde çalıştırmak için:

1. Projeyi klonlayın:
```bash
git clone https://github.com/semihdayioglu/barber-shop-website.git
```

2. Proje dizinine gidin:
```bash
cd barber-shop-website
```

3. `index.html` dosyasını bir web tarayıcısında açın.
